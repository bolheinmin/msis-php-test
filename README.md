
# MSIS PHP - Laravel Test 




## Run Locally

Clone the project

```bash
  git clone https://github.com/bolheinmin/msis-php-test.git
```

Go to the project directory

```bash
  cd msis-php-test
```

Install dependencies

```bash
  composer install
```
Database Seeder
```bash
  php artisan migrate --seed
```

Start the server

```bash
  php artisan serve
```
